## What Admin can do
- [x] CRUD Vehicles
- [x] Assign Vehicles to Drivers
- [x] Schedule Maintenance for Vehicles
- [x] Assign Maintenance Tasks to Mechanics
- [x] View Reports and Analytics
- [x] Manage User Roles and Permissions

- [x] View Assigned Vehicles

- [x] Update Personal Profile and Contact Information

- [x] View Assigned Maintenance Tasks
- [x] Update Maintenance Task Status (e.g., In Progress, Completed)
- [x] Log Maintenance Details
- [x] View Maintenance History of Vehicles
- [x] Update Personal Profile and Contact Information
